### GET /api/tag/all/:include
Получение списка тегов из БД в порядке релевантности.<br/>
/api/all - вернет все теги из базы
```
[
    "tag1",
    "tag3",
    "dsds",
    "спасение",
    "животные",
    "Москва",
    "дети",
    "лекарства"
]
```
?limit=3 - установит лимит в 3 тега
```
[
    "tag1",
    "tag3",
    "dsds"
]
```
/api/all/ва - вернет теги, которые содержат "ва"
```
[
    "Москва",
    "лекарства"
]
```
